# 流浪动物救助与领养平台

## 项目简介
为校园流浪动物救助社团开发的信息化管理平台，解决救助信息分散、领养流程不规范的问题。实现动物信息发布、领养申请审批、图片上传等核心功能。

## 技术栈
- 前端：Vue 3 + Element Plus + Vue Router + Axios
- 后端：Spring Boot + MyBatis-Plus + Maven
- 数据库：MySQL 8.0
## 功能模块
-用户注册/登录（JWT认证）
-动物信息的增删改查（支持图片自行上传）
-领养申请提交与审批
-管理员后台（用户管理、数据统计）
